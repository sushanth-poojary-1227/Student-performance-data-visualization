<h2>Student Performance Data Visualization</h2>
Description

This project explores student academic performance using data visualization techniques.
The goal is to visually understand patterns and trends related to student performance, not to predict or evaluate individual students.

Dataset

Student Performance Dataset (UCI / Kaggle)

File used: student-mat.csv

Tools

Python

Jupyter Notebook

Pandas, Matplotlib, Seaborn

Visualizations

Study time vs final grade

Past failures vs performance

Absences vs performance

Gender vs performance

Note

This project focuses on exploratory data visualization only.
